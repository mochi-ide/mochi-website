# MOCHI Website

This folder contains the landing page for MOCHI - Machine Output Checker & Heuristic Intelligence.

## How to View
Simply open `index.html` in your web browser.

## Structure
- `index.html`: Main landing page.
- `style.css`: Stylesheet with dark mode and animations.
- `script.js`: Interactivity and scroll effects.
- `assets/`: Images and icons.

## Features Highlighted
- Smart Log Analysis
- LLM Chat Context
- State Machine Visualization
- Codebase Embeddings
